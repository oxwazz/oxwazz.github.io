## Pendahuluan
Mungkin langsung aja karena memang caranya cukup mudah, seperti:

1. kita buka terminal lalu ketikkan `upower -i` untuk ngelist semua devices yang ada pada *Upower*
```bash
❯ upower -i

  /org/freedesktop/UPower/devices/battery_BAT1
  /org/freedesktop/UPower/devices/line_power_ACAD
  /org/freedesktop/UPower/devices/battery_hidpp_battery_0
  /org/freedesktop/UPower/devices/DisplayDevice
```

2. kemudian kita cari path yang ada tulisan *BAT*, dikasus ini adalah `/org/freedesktop/UPower/devices/battery_BAT1`, setelah kita dapat path tersebut, kita ketikkan: [^link1]
```bash
❯ upower -i /org/freedesktop/UPower/devices/battery_BAT1`

  native-path:          BAT1
```
